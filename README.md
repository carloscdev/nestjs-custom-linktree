# CUSTOM LINKTREE

1. Clone Project
```bash
  git clone https://github.com/....
```
2. Install dependencies
```bash
  yarn
```
3. Create .env file and configure with values
4. Database configuration
```bash
  docker-compose up -d
```
5. Run project
```bash
  yarn start:dev
```
